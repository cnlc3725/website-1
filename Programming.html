This is a comprehensive guide to programming with different sections for kids 5-8, 9-13, and 14+.

Blocks

Blocks is the simplest coding language, where you drag blocks together and access drop-down menus to change parts of the block. It is color-coded, which can be an eyesore to some, but is perfect for 5-8 year olds(or younger). If they have not used blocks before, I recommend Scratch JR. For people who have used blocks before, i recommend scratch. I personally started with scratch, and even made my own projects for fun that have been uploaded. Both scratch versions and all of my projects work on any platform.

Python

Python is a more complicated language compared to blocks, but is a great introduction to text coding. It is relatively easy to use, and is one of the languages with the most restrictions in it. It is built for basic website building, automation, and programming robots. I used it to program our VEX V5RC robot and it was simple to learn, even after not using it for too long. I recommend using it with the python extension in Microsoft visual studio Code to make basic programs to automate certain tasks on your computer that are repetitive, or you can have it do random actions and see what it can make in an app.
